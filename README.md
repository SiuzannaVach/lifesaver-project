# Página de aterrizaje Lifesaver IKIGAI

Este proyecto es una página web estática tipo *landing page* llamada Lifesaver.

## Estructura del proyecto

- `index.html` — página principal
- `styles.css` — archivo principal de estilos
- `header.css`, `hero.css`, `features.css`, `footer.css` — archivos de estilos por secciones
- `assets/` — imágenes y recursos adicionales

## Cómo ejecutar

1. Abrir el proyecto en Visual Studio Code.
2. Instalar la extensión **Live Server**.
3. Hacer clic derecho sobre `index.html` → **Open with Live Server**.
4. La página se abrirá en tu navegador para ver los cambios en tiempo real.

## Notas

- La página es **responsive**, se adapta a diferentes tamaños de pantalla.
- Se utilizaron **HTML5** y **CSS3**, incluyendo `@import` para secciones.
- Para cualquier cambio en los estilos, editar los archivos CSS correspondientes.
